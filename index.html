從只能輸入文字的單欄位，升級為可預覽、可下拉選參數的雙模輸入體驗。

改版重點

  • 左側即時預覽，輸入後立刻看到顏色，不需腦中換算。
  • 下拉選單彙整現有設計參數（assist / primary / functional），降低記憶成本。
  • 參數與 Hex 雙向支援：選參數會自動填入 Hex，反之亦然。
  • 狀態清楚：預設、Hex 輸入中、已選參數三種樣式區分明確。
  • 必填驗證：點擊輸入框後離開若為空，立即顯示紅框與「不得為空」提示。
  • 輸入框自動帶入 #，並限制最多 6 碼 Hex 字元，避免格式錯誤。
  • 智能展開：輸入 1 碼 F 自動補成 #FFFFFF、2 碼 14 → #141414、3 碼 123 → #123123，加快常用色輸入。
  • 已有 Hex 時，再次聚焦會自動全選，方便直接覆寫不必手動清除。
  • 參數綁定可解除：選參數後 token chip 旁有 ×，按下會保留 Hex 但解除與設計參數的連結，方便切換成寫死色碼。
Component Redesign

displayPattern 改版

從讓使用者自由輸入格式字串，改為清楚的下拉選單，並針對小數提供位數設定。

原版

不得為空

只能輸入文字，無格式驗證；使用者得自己記住「字串 / 整數 / 0.0 / 0.00」之類的規則，容易輸錯。

改版重點

  • 三個明確選項（字串／整數／小數）取代自由輸入，使用者不必背格式規則。
  • 選擇「小數」會額外帶出「取到小數第 N 位」的計數器，把原本要寫死在格式字串裡的位數抽成獨立欄位。
  • 必填驗證：點過後若沒選會顯示紅框與「不得為空」提示。
  • 下拉展開時 chevron 會旋轉，提供清楚的開合視覺回饋。
